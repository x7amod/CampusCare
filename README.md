## Project Setup Instructions

1. Install Xcode version **16.4+**.  
2. Clone the project using GitHub Web or CLI:
3. Open the project in Xcode.
4. Select the iPhone 16 simulator.
5. Run the app using the Run button at the top.

The app should build and launch in the simulator.
